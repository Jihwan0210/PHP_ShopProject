PHP Shop Project
프로젝트 소개
이 프로젝트는 PHP와 MySQL을 사용한 간단한 쇼핑몰 예제입니다. 상품 목록, 장바구니, 주문, 회원 관리, 관리자 페이지 등의 기능을 포함하고 있습니다.

주요 기능
상품 목록 조회 및 검색
장바구니 추가/수정/삭제
주문 입력 및 결제 페이지
회원 가입, 로그인, 정보 수정
관리자 상품 등록/수정/삭제
주소 검색, 문의 게시판, FAQ
설치 방법
로컬 서버(XAMPP, WAMP 등)를 준비합니다.
프로젝트 폴더를 웹 루트 디렉터리에 복사합니다.
데이터베이스를 생성하고 필요한 테이블을 구성합니다.
common.php, common.php, common.php 파일의 DB 연결 정보를 본인 환경에 맞게 수정합니다.
사용 시 주의
common.php, common.php, common.php 파일에는 데이터베이스 정보가 포함되어 있으므로 공개 저장소에 민감 정보를 올리지 않도록 주의해야 합니다.
실제 서비스용으로 배포할 때는 비밀번호, 관리자 계정 정보 등을 별도 설정 파일이나 환경 변수로 관리하는 것이 좋습니다.
개발 환경
PHP
MySQL
HTML/CSS/JavaScript
Bootstrap
추가 정보
이 저장소는 학습용 및 포트폴리오용으로 작성되었습니다.
실제 운영 환경에서는 보안과 입력 검증을 강화해야 합니다.
