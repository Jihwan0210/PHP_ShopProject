## 목적
PHP 기반 쇼핑몰을 개발하고 상품 조회, 장바구니, 주문 기능 및 관리자 상품 관리 기능을 구현하며 전자상거래 서비스 개발 프로세스를 경험하기 위함입니다.

---

## 설명
이 프로젝트는 PHP와 Bootstrap을 활용해 회원가입, 로그인, 상품 목록/상세 페이지를 구현하고, 쿠키를 이용해 로그인 정보 및 비회원 장바구니 기능을 적용하였습니다.

주문 과정에서는 옵션 선택, 수량 조절, 할인율 및 배송비 계산 기능을 구현하였으며, 결제 완료 시 주문번호와 주문 내역이 저장되도록 구성했습니다.

관리자 페이지에서는 상품 등록·수정·삭제, 이미지 업로드, 옵션(사이즈·색상) 관리 기능을 제공하며, 데이터베이스 관리는 phpMyAdmin으로 수행하였고 개발 도구는 Notepad++를 사용했습니다.

---

## 주요 기능
- 상품 목록 조회 및 검색
- 장바구니 추가, 수정, 삭제
- 주문 입력 및 결제 페이지
- 회원 가입, 로그인, 정보 수정
- 관리자 상품 등록, 수정, 삭제
- 주소 검색, 문의 게시판, FAQ

---

## 설치 방법
1. 로컬 서버(XAMPP, WAMP 등)를 준비합니다.
2. 프로젝트 폴더를 웹 루트 디렉터리에 복사합니다.
3. 데이터베이스를 생성하고 필요한 테이블을 구성합니다.
4. `html/common.php`, `html/juso/common.php`, `html/sj/common.php` 파일의 DB 연결 정보를 본인 환경에 맞게 수정합니다.

---

## 사용 시 주의
- `html/common.php`, `html/juso/common.php`, `html/sj/common.php` 파일에는 데이터베이스 정보가 포함되어 있으므로 공개 저장소에 민감 정보를 올리지 않도록 주의해야 합니다.
- 실제 서비스용으로 배포할 때는 비밀번호와 관리자 계정 정보를 별도 설정 파일이나 환경 변수로 관리하는 것이 좋습니다.
